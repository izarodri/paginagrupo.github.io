HTML é uma linguagem de programação?

Ao contrário do que muitas pessoas que estão começando a estudar a linguagem pensam, HTML não é uma linguagem de programação. HTML é uma sigla para: Hypertext Markup Language, que por si já entrega seus objetivos se traduzida para o português, que significa Linguagem de Marcação de Hipertexto. Apesar do nome complicado, HTML nada mais é do que uma linguagem usada para criar páginas web usando tags e atributos que definem como o conteúdo deve ser apresentado em um navegador da web. Essa linguagem é responsável por comandar tudo, e neste caso, não funcionam como instruções, como toda linguagem de programação.

O termo “programação” engloba características primordiais que não estão presentes no HTML, como o uso de variáveis simples e compostas, condições, laços e coisas com mais complexidade, como objetos, métodos e classes que não estão presentes na HTML e nem na CSS. Em suma, HTML trabalha apenas nas marcas ou etiquetas (tags) e as CSS funcionam com base nos setores, propriedades e valores.

Ainda falando sobre o significado da sigla HTML, o que seria um Hipertexto? O conceito se refere a um texto com blocos interconectados contendo palavras, imagens, tabelas videos entre outros elementos. Eles também podem ter caminhos que apontam para outros hipertextos, conhecidos como hiperligações, hyperlinks ou apenas links.

Para que serve HTML, CSS e JavaScript?

Uma das coisas mais importantes para quem está começando o desenvolvimento de sites é compreender para que serve esse trio de tecnologias, que geralmente são estudados em conjunto. Basicamente, de forma resumida, temos um panorama simples:

HTML Conteúdo
CSS Estilo
JavaScript Interatividade

Conseguimos compreender que o HTML é a estrutura de uma página web. Para definir o estilo de seus elementos, como fontes, cores, alinhamento, basicamente a forma como os componentes HTML serão apresentados, é utilizado o CSS, sigla para Cascading Style Sheets, ou folhas de estilo em cascata.

Por fim temos o JavaScript. Essa, que sim é uma linguagem de programação, e nos permite dar mais interatividade à página, com fim de melhorar a experiência do usuário.

Uma ótima forma de distinguir isso bem, é abrindo um site qualquer. Observe os textos, as imagens, videos, links, tabelas e todo o conteúdo agregado na página. tudo isso é produzindo em HTML, logo a linguagem é focada nos conteúdos.

Agora foque nas cores, nas fontes dos textos, nos formatos das imagens, margens, e toda a organização dos conteúdos da página. Isso foi tudo estabelecido em CSS, pois é focada no design.

CURIOSIDADES HTML

  • HTML é acrônimo de HyperText Markup Language: A sigla HTML significa HyperText Markup Language, e foi criada em 1990 por Tim Berners-Lee, que também é conhecido como o "Pai da Web".
  • HTML evoluiu ao longo do tempo: Desde a sua criação, o HTML tem passado por diversas atualizações e hoje em dia utilizamos a versão HTML5, que é a mais recente.
  • HTML é uma linguagem de marcação baseada em tags: As tags são elementos fundamentais da linguagem HTML, usados para indicar ao navegador como o conteúdo deve ser exibido na página.
  • HTML permite criar páginas acessíveis: Uma das grandes vantagens do HTML é a sua capacidade de criar páginas acessíveis, que podem ser lidas por pessoas com deficiências visuais ou motoras.
  • HTML pode ser usado para desenvolver aplicativos: Além de ser usada para desenvolver páginas web, a linguagem HTML também pode ser utilizada para criar aplicativos, principalmente com o auxílio de frameworks como o React Native.
  • HTML é importante para o SEO: Utilizar a linguagem HTML corretamente é fundamental para otimizar o posicionamento de um site nos resultados de busca, tornando-o mais visível e acessível para os usuários da web.
