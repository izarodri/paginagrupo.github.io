O Que São Tags HTML?

As tags são utilizadas para informar ao navegador a estrutura do site. Elas serão interpretadas pelo navegador, estabelecendo a estrutura e o conteúdo visual da página. Elas são caracterizadas por estarem sempre dentro dos sinais “maior que” e “menor que” (<>).

As tags que precisam de fechamento possuem a sintaxe <tag> </tag> , e as que não possuem fechamento Uma única tag pode receber varios atributos, que terá um valor que poderá modificar sua estrutura ou funcionalidade.

Atributos HTML

Os atributos são os responsáveis por personalizar as tags, e são utilizados para atribuir uma classe ou id a algum elemento. As tags possuem seus próprios atributos no geral, porém existem atributos genéricos que são usados em grande parte das tags. Observe na tabela:

NOME DO ATRIBUTO ELEMENTOS FUNCIONALIDADES
class Global attribute Usado com CSS para estilizar os elementos com propriedades comuns.
id Global attribute Usado com CSS para estilizar um elemento específico. O valor deste atributo deve ser único.
style Global attribute Permite a utilização de elementos de estilos CSS dentro das tags.
lang Global attribute Define o idioma usado no elemento.
title Global attribute Define o título do elemento.
alt <applet>, <area>, <img>, <input>, Texto alternativo caso uma imagem não possa ser exibida.
hidden Global attribute Indica a relevância de um elemento.
align <applet>, <caption>, <col>, <colgroup>, (en-US), <hr>, <iframe>, <img>, <table >, <tbody>, (en-US), <td>, (en-US), <tfoot>, <th>, <thead>, (en-US), <tr>, Especifica o alinhamento horizontal do elemento. Exemplos: right, center, left e justify.
width <canvas>, <embed>, <iframe>, <img>, <input>, <object>, (en-US), <video>. Define uma largura para o elemento.
height <canvas>, <embed>, <iframe>, <img>, <input>, <object>, (en-US), <video>. Define uma altura para o elemento.

Vale ressaltar que estes são apenas alguns dos varios atributos existentes que podem ser implementados ao código de acordo com cada necessidade.
