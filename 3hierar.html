Tags de Título

Utilizar as tags de forma adequada é fundamental para organizar o conteúdo da página de forma clara e semântica. Existem duas tags HTML que são utilizadas para definir o título de uma página: a tag <title> e a tag <h1>.

A tag <title> é utilizada dentro do cabeçalho da página, entre as tags <head> e </head>. O texto inserido dentro desta tag será exibido na aba ou janela do navegador, e é importante para o SEO da página, pois é utilizado pelos motores de busca para indexar o conteúdo. Segue o exemplo abaixo:

                    
<!DOCTYPE html>
<html>
  <head>
    <title>Título da minha página</title>
  </head>
  <body>
    ...
  </body>
</html>

Já a tag <h1> é utilizada para definir o título principal da página, geralmente no início do conteúdo principal. Ela é utilizada para indicar ao usuário e aos motores de busca qual é o tema principal da página. Segue o exemplo abaixo: 

                
<!DOCTYPE html>
<html>
  <head>
    <title>Título da minha página</title>
  </head>
  <body>
    <h1>Título principal da minha página</h1>
     ...
  </body>
</html>

Vale ressaltar que esta tag pode variar de tamanho de acordo com o número colocado nela da seguinte forma: 

                
<h1></h1> -  Título de maior valor hierárquico

 <h2></h2>
 <h3></h3>
 <h4></h4>
 <h5></h5>
 <h6></h6> - Título de menor valor hierárquico

Tags de Texto

As tags de conteúdo textual são responsáveis por definir elementos de texto, fontes, parágrafos, spans, além de permitir quebras de linha e outras formatações. Abaixo, apresentamos algumas das principais tags utilizadas para esse fim:

  • <p></p>– Principal tag de texto, compõe um parágrafo;
  • <span></span>– Apesar de ter uma funcionalidade e características parecidas com os parágrafos, costumam ser utilizadas apenas para pequenas informações, como legendas de um formulário, legendas de uma imagem, etc. Também pode ser utilizada para formar um container;
  • <pre></pre>– Tag utilizada para representar texto pré-formatado. Muito utilizada para inserir códigos;
  • <b></b>– Transforma o conteúdo em negrito;
  • <i></i>– Transforma o conteúdo em itálico;
  • <br/>– Essa tag não necessita de fechamento, ela executa a função de quebra de linha.
  • <hr/> – Essa tag não necessita de fechamento, ela forma uma linha horizontal.

Tag para Inserir links

A tag HTML para hiperlink tem como função conectar diferentes documentos, tanto na mesma página quanto em outras páginas de diferentes domínios. Esta tag é considerada uma das bases mais importantes para a estrutura da web como a conhecemos hoje em dia.

Para criar um link em seu documento HTML, você pode utilizar a tag <a></a> e o atributo "href". Por exemplo, se você quiser adicionar um link em seu texto que redirecione para a página inicial do Google: 

                
<p>Para acessar o Google, 
      <a href="https://www.google.com">clique aqui.</a></p>

Resultado na página:

Para acessar o Google, clique aqui.

Tags para Inserir Multimídias

As tags de multimídias servem para colocar mídias na página, como imagens, vídeos, áudios, iframes e outros tipos de mídias. A seguir estão as principais tags utilizadas:

  • <img> - Esta tag é utilizada para inserir imagens em um documento HTML. Ela é um elemento vazio, ou seja, não possui um elemento de fechamento, e utiliza o atributo "src" para especificar o caminho do arquivo de imagem que será exibido na página. Além disso, a tag também permite a inclusão de outros atributos, como "alt" para adicionar uma descrição da imagem e "width" e "height" para definir as dimensões da imagem exibida na página.

    Exemplo:

                            
<img src="imagens/imagem1.jpg" 
         alt="imagem 1 da minha página">

  • <video> - É utilizada para inserir vídeos em um documento HTML. Ela suporta diferentes formatos de arquivos de vídeo, como MP4, WebM e Ogg, e permite a inclusão de controles de reprodução, como botões de play, pause e volume. Além disso, a tag <video> também permite a definição de atributos, como "width" e "height" para especificar as dimensões do vídeo na página, "poster" para adicionar uma imagem de preview do vídeo e "autoplay" para definir se o vídeo deve ser reproduzido automaticamente quando a página é carregada.

    Para permitir o controle de reprodução do vídeo, é necessário incluir alguns atributos na tag <video>, sendo o mais importante deles o "controls". Em seguida, é preciso adicionar a tag <source> com os atributos "src" e "type", que especificam o local e o tipo do arquivo de vídeo, respectivamente. É recomendável incluir várias opções de source para o mesmo vídeo, a fim de garantir compatibilidade com diferentes navegadores. Por fim, é possível adicionar uma mensagem personalizada para os casos em que o navegador não suporta a tag <video>.

    Exemplo:

                                 
<video controls>
            <source src="video.mp4" type="video/mp4">
            <source src="video.ogg" type="video/ogg">
            Seu navegador não possui suporte para Vídeos.
</video>

  • <audio> - A tag <audio> funciona da mesma forma que a tag <video>, portanto, é utilizada a tag dentro dela.

    Exemplo:

                            
<audio controls>
            <source src="musica.ogg" type="audio/ogg">
            <source src="musica.mp3" type="audio/mpeg">
            Seu navegador não possui suporte para áudio.
</audio>

  • <iframe> - É utilizada para incorporar conteúdo de outras fontes em uma página HTML. Por exemplo, é possível incorporar um mapa do Google Maps, um vídeo do YouTube ou até mesmo uma página inteira de outro site. Para isso, basta especificar o URL do conteúdo a ser incorporado no atributo "src" da tag <iframe>. Além disso, é possível definir outras propriedades, como altura, largura e borda, para personalizar a aparência do iframe. No entanto, é importante ter cuidado ao incorporar conteúdo de fontes externas, pois pode haver riscos de segurança e de privacidade.

    Exemplo:

                            
<iframe src="https://www.homehost.com.br">
            <p>Seu navegador não possui suporte para iFrames</p>
</iframe>

Tags para Adicionar Listas

Para criar uma lista em HTML, podemos utilizar uma lista ordenada, ou uma lista não ordenada, e também uma lista de definição. Em seguida, dentro da lista, podemos incluir os elementos das listas.

  • Lista ordenada: <ol> (abre a lista ordenada) e </ol> (fecha a lista ordenada). Cada item da lista é indicado pela tag <li>, que deve ser colocada dentro das tags <ol> e </ol>

    Exemplo:

                            
<p>Minha lista ordenada:</p>
        <ol>
            <li>item 1</li>
            <li>item 2</li>
            <li>item 3</li>
        </ol>

  • Lista não ordenada: <ul> (abre a lista não ordenada) e </ul> (fecha a lista não ordenada). Cada item da lista também é indicado pela tag <li>, que deve ser colocada dentro das tags <ul> e </ul>.

    Exemplo:

                            
<p>Minha lista não ordenada:</p>
        <ul>
            <li>item 1</li>
            <li>item 2</li>
            <li>item 3</li>
        </ul>

  • Lista de definição: <dl> (abre a lista de definição) e </dl> (fecha a lista de definição). Cada item da lista é composto por dois elementos: um termo, indicado pela tag <dt>, e sua definição, indicada pela tag <dd>. Ambas devem ser colocadas dentro das tags <dl> e </dl>.

    Exemplo:

                             
<dl>
    <dt>linguagem 1</dt>
        <dd>HTML (HyperText Markup Language)</dd>
     <dt>linguagem 2</dt>
        <dd>CSS (Cascading Style Sheets)</dd>
    <dt>linguagem 3</dt>
            <dd>JS (JavaScript)</dd>
</dl>

Tags de estilos e scripts

Por fim, duas tags que não podem ser deixadas de lado: <style> e <script>

A tag <style> deve ser inserida no cabeçalho do documento HTML, dentro das tags <head></head>. Nela, é possível incluir todo o seu código CSS, que define os estilos de sua página

Já a tag <script> é usada para incluir código JavaScript em seu HTML e pode ser inserida em qualquer parte do documento, embora seja recomendado colocá-la antes da tag de fechamento </body>. Assim, você pode incluir nela seu código JavaScript para tornar sua página interativa e dinâmica.
