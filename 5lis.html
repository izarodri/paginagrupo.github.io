Tags para Inserir Tabelas

A estrutura básica de uma tabela HTML é composta por três elementos principais: a tag <table> que indica o início da tabela, a tag </table>que indica o final da tabela, e as células da tabela que são definidas pelas tags <tr> e <td>.

A tag <tr> é usada para definir cada linha da tabela, enquanto a tag <td> é usada para definir cada célula da tabela. Dentro das tags <td>, podemos adicionar o conteúdo que será exibido em cada célula da tabela. Além disso, podemos usar outras tags para formatar e estilizar a tabela, como a tag <th> para cabeçalhos de coluna, as tags <thead>, <tbody> e <tfoot> para definir as partes da tabela, e a tag <caption> para adicionar um título à tabela.

Abaixo está um exemplo: 

                
<table border="1">
    <tr>
        <th>Nome</th>
        <th>Idade</th>
        <th>Profissão</th>
    </tr>
    <tr>
        <td>Dan</td>
        <td>16</td>
        <td>Estudante</td>
    </tr>
    <tr>
        <td>Iza</td>
        <td>26</td>
        <td>Designer</td>
    </tr>
</table>

O resultado disso é o seguinte:

Nome Idade Profissão
Dan 16 Estudante
Iza 26 Designer

Também é possivel o <th> para títulos verticais, ou seja, na primeira coluna. Dessa forma, vamos para o seguinte exemplo de tabela HTML:

                        
<table border="1">
   <tr>
       <th>Nome:</th>
       <td>Eli</td>
   </tr>
   <tr>
       <th>Idade:</th>
       <td>45</td>
   </tr>
</table>
O resultado deste exemplo será:

O resultado deste exemplo será:

Nome: Eli
Idade: 45

Quando se deseja inserir uma célula vazia em uma tabela HTML, é comum usar a tag <td> ou <th> sem adicionar qualquer conteúdo. Entretanto, essa abordagem pode não ser recomendada, pois alguns navegadores podem não conseguir ler adequadamente uma célula vazia. A solução mais apropriada para essa situação é utilizar a técnica de espaço, que consiste em adicionar a entidade   dentro da célula. Isso criará um espaço em branco dentro da célula e evitará problemas de compatibilidade com diferentes navegadores. 

                
<table>
    <tr>
        <th>&nbsp;</th>
        <th>Valor</th>
    </tr>
    <tr>
        <td>Item 1</td>
        <td>R$10,00</td>
    </tr>
    <tr>
        <td>Item 2</td>
        <td>R$20,00</td>
    </tr>
</table>

Resultado:

  Valor
Item 1 R$10,00
Item 2 R$20,00

Em certos casos, precisamos criar uma célula que abranja mais de uma linha ou coluna em uma tabela HTML. Para isso, podemos usar os atributos colspan e rowspan. O atributo colspan="x" permite mesclar colunas, enquanto o atributo rowspan="x" permite mesclar linhas. Basta substituir o "x" pelo número de colunas ou linhas que se deseja ocupar. Além disso, é possível mesclar ambos os atributos em uma única célula.

Exemplo utilizando o colspan: 

                
<table border="1">
    <tr>
        <th>Nome</th>
        <th colspan="2">Telefones</th>
    </tr>
    <tr>
        <td>JP</td>
        <td>1111-1111</td>
        <td>2222-2222</td>
    </tr>
    <tr>
        <td>Iza</td>
        <td>3333-3333</td>
        <td>4444-4444</td>
    </tr>
</table>

Resultado:

Nome Telefones
JP 1111-1111 2222-2222
Iza 3333-3333 4444-4444

Exemplo utilizando o rowspan:

                    
<table border="1">
<tr>
  <th>Primeiro Nome:</th>
  <td>Ted</td>
</tr>
<tr>
  <th rowspan="2">Telefone:</th>
  <td>8888-8888</td>
</tr>
<tr>
  <td>9999-9999</td>
</tr>
</table>

Resultado:

Primeiro Nome: Ted
Telefone: 8888-8888
9999-9999

Para incluir um título em uma tabela HTML, utilizamos a tag <caption> </caption>. Ao contrário dos demais elementos da tabela, a tag <caption> não é composta por células e, portanto, fica localizada acima da tabela, como um título descritivo.

Exemplo:

                        
 <table border="1">
    <caption>Exemplo de Título</caption>
    <tr>
        <th>Nome</th>
        <th>Sobrenome</th>
    </tr>
    <tr>
        <td>Iza</td>
        <td>Bela</td>
    </tr>
    <tr>
        <td>Dan</td>
        <td>Silva</td>
    </tr>
</table>

Resultado:

Exemplo de Título
Nome Sobrenome
Iza Bela
Dan Silva

O Colgroup e Col

Muitos dos atributos de tabela em HTML são agora considerados obsoletos. Consequentemente, a formatação e estilização de tabelas deve ser realizada por meio de CSS. Para esse propósito, existe um elemento-chave chamado <colgroup></colgroup>, que permite a criação de um grupo de colunas. Dessa forma, várias colunas podem ser estilizadas de uma só vez. Para definir cada grupo de coluna, utilizamos as tags </col> .

Exemplo aplicando o CSS às classes que utilizaremos nas <col> :

                
 <style>
    .vermelho{
        background-color: red;
    }
    .amarelo{
        background-color: yellow;
    }
    .azul{
        background-color: blue;
    }
</style>

<table border="1">
  <colgroup>
    <col class="vermelho">
    <col class="amarelo">
    <col class="azul">
  </colgroup>
  <tr>
    <th>Nome</th>
    <th>Idade</th>
    <th>Profissao</th>
  </tr>
  <tr>
    <td>JP</td>
    <td>22</td>
    <td>Estudante</td>
  </tr>
  <tr>
    <td>Eli</td>
    <td>26</td>
    <td>Escritor</td>
  </tr>
</table>

Resultado:

Nome Idade Profissao
JP 22 Estudante
Eli 26 Escritor
